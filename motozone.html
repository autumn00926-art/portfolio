한 줄 요약
MOTOZONE은 차량 정보를 빠르게 탐색하고 비교할 수 있도록 UI 구조와 콘텐츠 흐름을 설계한 자동차 플랫폼 웹 프로젝트입니다.
UX Platform Automotive
기획 의도

기존 자동차 플랫폼은 정보가 많아 사용자가 원하는 차량에 도달하기까지 피로도가 높다고 느꼈습니다.

MOTOZONE은 추천 → 비교 → 선택 흐름을 명확히 하여 사용자가 자연스럽게 탐색할 수 있는 구조를 목표로 했습니다.

시각적으로는 신뢰감을 주는 톤을 유지하면서, 핵심 정보가 빠르게 인지되도록 UI를 구성했습니다.

Design Strategy
  • Layout
    카드 기반 레이아웃으로 정보 인지 속도 개선

    Color
    블루 계열 포인트 컬러로 신뢰감 강화

    Interaction
    버튼·슬라이드 중심의 직관적인 인터랙션

차량 이미지가 가장 먼저 인지되도록 구성하고, 가격·차종·특징 정보를 단계적으로 노출했습니다.

이를 통해 사용자가 탐색 과정에서 이탈하지 않고 자연스럽게 다음 행동으로 이어지도록 설계했습니다.

회고 · 배운 점

이후 웹 작업에서도 사용자의 행동을 먼저 설계한 뒤 시각 디자인을 구성하는 방식으로 접근하고 있습니다.
